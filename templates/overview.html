Oauth for Apache Cordova

An AngularJS Apache Cordova Oauth library. The purpose of this library is to quickly and easily obtain an access token from various web services to use their APIs.

Multiple Providers

Support for popular service providers including Facebook, Twitter, and Google.

Hassle Free

Get an access token for any of the supported providers with as little as three lines of code.

Open Source

The full source code can be forked on GitHub for free under the MIT license.
